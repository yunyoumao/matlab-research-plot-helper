# Code of Conduct

This project is for practical, public-safe research plotting workflows.

Participants are expected to:

- be respectful and constructive
- avoid uploading private or unpublished data
- use synthetic or public-domain examples only
- focus feedback on clarity, reproducibility, and maintainability

Maintainers may edit, hide, or remove content that exposes private information, unpublished research assets, or unsafe contribution material.
