# Contributing

Contributions are welcome if they keep the repository public-safe and dependency-light.

Good contributions include:

- improved MATLAB plotting defaults
- additional synthetic examples
- export compatibility fixes
- documentation improvements
- validation checks for public assets

Before submitting a change, check that:

- all data are synthetic
- no real experiment, test-rig, manuscript, or company data are included
- no local absolute paths, private emails, phone numbers, or credentials are included
- demos run from the repository root
- generated figures remain readable after resizing

Please avoid adding toolbox-specific dependencies unless the fallback remains usable without that toolbox.
